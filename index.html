What is Cardify 2.0?

Cardify 2.0 is a jQuery plugin that makes easy to add a simple hover effect to your images.

Install

a. Local installation

To install cardify 2.0 locally in your project follow the next steps:

1. Add jQuery in your project

2. Add cardify.css in your css folder and write the path in your html

3. Add cardify.js in your js folder and write the path in your html

4. Add a container with class="imgwrap" in your html

b. Global installation

To install cardify 2.0 by CDN in your project add these link and script:

1. Add jQuery in your project

2. Add CDN in your proyect

Use

Users can add cardify 2.0 in their projects at local and globally way. For local path they can to download css and js files fron github link. For global path they can add CDN links.
